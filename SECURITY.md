The rottingresearch project takes security seriously.

To report a security vulnerability, please use the **Security Vulnerability Report** issue template.

## Responsible Disclosure

- Please do not publicly disclose vulnerabilities before maintainers have reviewed them.
- For critical issues, maintainers may request private coordination before public discussion.

## Severity Levels

| Level | Description |
|------|------------|
| Low | Minor issue, no direct data exposure |
| Medium | Limited impact under specific conditions |
| High | Significant impact such as data exposure or auth bypass |
| Critical | Remote exploitation or system compromise |

## Response Expectations

Maintainers aim to:
- Acknowledge reports within **72 hours**
- Provide updates as mitigation progresses
- Credit reporters when appropriate

Thank you for helping keep rottingresearch secure.
